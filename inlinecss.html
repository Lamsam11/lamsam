Cascading Style Sheet(css)

Cascading is used to style and describe the layout on a web page

CSS can be added in a web page via three methods below

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Advantages of CSS

  • changing the color of the element
  • Adjusting the size and dimension
  • Adjusting the layout of element
